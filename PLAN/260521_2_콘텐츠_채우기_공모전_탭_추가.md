# 1단계 콘텐츠 채우기 + 공모전 탭 추가

## Context

1단계 골격이 완성된 직후, 사이트가 비어 보이지 않도록 더미 콘텐츠를 풍부하게 채운다. 동시에 "공모전 정보"(`/contests`) 탭을 신설해 모임 멤버가 함께 도전할 만한 공모전을 정리한다. 운영자/팀장 이름은 **제이쏭**으로 반영한다.

## 결정사항

- **운영자(팀장)**: 제이쏭. About 페이지·푸터·프로젝트 멤버에 반영.
- **첫 실제 프로젝트**: "오늘 뭐 먹지?" — https://what-to-eat-today-sand.vercel.app/ — 메뉴 추천 사이드 프로젝트.
- **첫 공모전**: 제4회 문화체육관광 인공지능·데이터 활용 공모전 (문화체육관광부 주최 · KCISA 한국문화정보원 주관, 2026-04-27 ~ 2026-06-26, 총 5천만원).
- **단계 로드맵 수정**: `/contests`를 원래 3단계 예정에서 **1단계로 앞당김**. CLAUDE.md의 로드맵·디렉토리 표기를 함께 갱신.
- 나머지(공지·일정·다른 프로젝트·다른 공모전)는 더미로 채우되, 자연스럽게 모임 활동이 진행 중인 듯한 분포로 작성.

## 데이터 스키마 (공모전)

```ts
type Contest = {
  id: string;
  title: string;
  host: string;       // 주최
  organizer?: string; // 주관
  startDate: string;  // YYYY-MM-DD
  endDate: string;    // YYYY-MM-DD
  prize: string;      // 시상 요약
  fields: string[];   // 분야 키워드
  eligibility: string; // 응모 자격 한 줄
  url: string;        // 공식 URL
  status: "모집중" | "예정" | "마감";
  tags: string[];
};
```

- 운영자 카드가 "제이쏭"으로 표시됨.
