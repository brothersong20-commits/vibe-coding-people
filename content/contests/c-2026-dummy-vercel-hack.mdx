Vercel Community Korea가 주최할 예정인 가상의 해커톤입니다. 모임의 주력 스택이 Next.js · Vercel · AI SDK 세트라, 평소 빌드 세션의 결과물을 그대로 출품해도 자연스럽게 맞아 떨어집니다.

## 왜 잘 맞는가

- 빌드 → 배포 사이클이 `git push`로 끝나는 환경이라 *마감 압박이 낮은 해커톤*이 됩니다.
- AI SDK Gateway로 모델만 갈아 끼우는 작은 데모가 점수 받기 좋은 영역입니다.
- 모임 입장에서는 출품 자체보다 "한 달 동안 하나를 끝내는 사이클"을 같이 돌려보는 것이 더 큰 수확입니다.

## 출품 후보 아이디어

```
1. /insights 페이지에 들어갈 AI 큐레이션 위젯을 따로 분리해 데모화.
2. 멤버용 코드 스니펫 공유 보드(Vibe Snippets)에 자연어 검색을 붙인 버전.
3. "오늘 뭐 먹지?"의 그룹 의사결정 모드 (링크 공유 → 다수결).
```

## 일정

- 접수: 2026-07-15 ~ 08-20
- 모임 일정: 7월 셋째 주에 킥오프 빌드 세션을 잡을 예정.
