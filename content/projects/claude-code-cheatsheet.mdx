Claude Code를 막 시작한 사람과 익숙한 사람이 동시에 빠르게 찾을 수 있는 단축키·슬래시 커맨드·훅 가이드를 한 페이지에 압축한 사이트.

## 채워둔 카테고리

- **단축키** — 명령어 모드, 코드 입력 모드 전환.
- **슬래시 커맨드** — `/help`, `/clear`, 그리고 자주 쓰는 커스텀 커맨드.
- **훅(Hook)** — 커밋 전·후, 도구 호출 전·후 자동화 패턴.
- **MCP 서버 설정** — 자주 쓰는 서버 3종 예시 설정.
